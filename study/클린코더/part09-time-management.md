# Part 9: 시간 관리 (Time Management)

## 핵심 내용

### 회의

#### 회의 거부하기
- 필요 없는 회의에는 참석하지 않는다
- 초대가 곧 의무는 아니다

#### 빠져 나오기
- 회의가 지루해지면 정중하게 나온다
- 시간 낭비를 허용하지 않는다

#### 의제와 목표를 정하라
- 명확한 의제 없는 회의에는 참석하지 않는다
- 회의의 목표를 분명히 한다

### 일일 회의 (Stand-Up Meeting)

**3가지 질문 (각 1분 이내):**
1. 어제 무엇을 했는가?
2. 오늘 무엇을 할 것인가?
3. 무엇이 방해가 되는가?

### 반복 계획 회의

- 짧게 유지한다
- 이터레이션 계획은 5~10분 내로

### 반복 회고와 시연

- 문제점을 빠르게 논의한다
- 시연은 간결하게

### 논쟁/의견 차이

- 5분 안에 해결되지 않으면 투표로 결정한다
- 끝없는 논쟁은 시간 낭비다

### 집중력 마나

집중력은 유한한 자원이다:
- **수면**: 충분히 자야 집중력이 회복된다
- **카페인**: 일시적 도움이 되지만 의존하지 마라
- **재충전**: 산책, 운동, 대화로 집중력을 회복한다

### 근육 집중

- 운동은 정신적 집중력 용량을 늘린다
- 규칙적인 운동이 필요하다

### 입력 vs 출력

- 창의력은 다양한 입력이 필요하다
- 책, 영화, 음악 등으로 영감을 얻는다

### 포모도로 기법

```
25분 집중 작업 → 5분 휴식 → 반복
4번 반복 후 → 15~30분 긴 휴식
```

### 우선순위 뒤집기

- 쉬운 일을 먼저 하려는 유혹에 넘어가지 마라
- 어려운 일을 먼저 한다
- 거짓말에 속지 않는다

### 막다른 길

- 잘못된 방향임을 깨달으면 빨리 돌아온다
- "구덩이의 법칙": 구덩이에 빠졌을 때 가장 먼저 할 일은 파기를 멈추는 것

### 진흙탕, 늪

- 초기 설계 실패를 방치하면 늪에 빠진다
- 빨리 빠져나와야 한다
- 리팩토링을 미루지 마라
