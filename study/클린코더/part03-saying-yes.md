# Part 3: 예라고 말하기

## 핵심 내용

### 약속을 뜻하는 말

진정한 약속의 3가지 구성요소:

1. **말한다 (Say)**: "나는 ~할 것이다"
2. **진심이다 (Mean)**: 정말로 그렇게 할 의도가 있다
3. **실행한다 (Do)**: 실제로 행동한다

### 약속하지 않는 표현들

다음과 같은 표현은 약속이 아니다:
- "해봐야 할 것 같다" / "해볼게요"
- "할 수 있으면 좋겠다"
- "~하는 게 좋을 것 같다"
- "아마도" / "~했으면 좋겠다"

### 진정한 약속의 표현

```
"나는 ~까지 ~을 완료할 것이다"
"화요일까지 이 기능을 완성하겠습니다"
```

### 예라고 말하는 법 익히기

#### 자신이 통제할 수 있는 일만 약속하라

```
잘못된 약속: "금요일까지 팀 전체가 이 기능을 완성할 것입니다"
올바른 약속: "금요일까지 나의 파트는 완성하겠습니다"
```

#### 불확실성을 인정하라

```
"현재 정보로는 3일 걸릴 것 같습니다.
하지만 X 조건이 변하면 더 걸릴 수 있습니다."
```

#### 약속을 지킬 수 없다면 즉시 알려라

- 문제가 생기면 가능한 빨리 알린다
- 새로운 약속을 제시한다

## 결론

- "예"라고 말하는 것은 강력한 약속이다
- 자신이 통제할 수 없는 일에는 약속하지 않는다
- 약속을 하면 반드시 지킨다
- 지킬 수 없다면 최대한 빨리 알린다
