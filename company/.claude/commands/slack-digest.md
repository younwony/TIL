# Slack Digest - 채널 대화 요약

특정 Slack 채널의 최근 대화 내용을 분석하여 핵심 내용을 요약해줘.

## 설정

| 항목 | 값 |
|------|------|
| **기본 요약 범위** | 최근 대화 (채널 읽기 기본값) |

## 1단계: 채널 결정

### `$ARGUMENTS`에 채널 지정이 있는 경우

- `$ARGUMENTS`에서 채널명 또는 채널 ID를 추출
- 예: `/slack-digest #dev-backend` → `dev-backend` 채널 요약
- 예: `/slack-digest C03D3F5DS02` → 해당 채널 요약

### 채널 지정이 없는 경우

AskUserQuestion으로 채널을 요청:
- "요약할 Slack 채널을 입력해주세요. (채널명 또는 ID)"
- 옵션:
  - "채널 검색" (검색어 입력)
  - "채널 ID 직접 입력"

### 채널 검색 시

도구: `mcp__claude_ai_Slack__slack_search_channels`
- 검색어로 채널 목록을 조회
- AskUserQuestion으로 채널 선택

## 2단계: 채널 대화 읽기

도구: `mcp__claude_ai_Slack__slack_read_channel`

- `channel_id`: 1단계에서 결정한 채널 ID
- `response_format`: "detailed"

### 채널 읽기 실패 시

- "채널을 읽을 수 없습니다. 채널 ID를 확인하거나, 채널에 접근 권한이 있는지 확인해주세요." 안내 후 중단

## 3단계: 대화 분석 및 요약

읽어온 대화 내용을 분석하여 요약한다.

### 분석 항목

- **주요 토픽**: 대화에서 다뤄진 주제들
- **핵심 메시지**: 중요한 공지, 결정, 변경사항
- **질의응답**: Q&A 형태의 대화 정리
- **액션 아이템**: TODO, 후속 작업
- **활발한 참여자**: 가장 많이 발언한 사람들

### 요약 템플릿

```markdown
## #{채널명} 대화 요약

**기간**: {시작 시간} ~ {끝 시간}
**총 메시지**: {N}건
**참여자**: {N}명

---

### 주요 토픽

1. **{토픽 1}**
   - {핵심 내용 요약}
   - 관련 참여자: {이름들}

2. **{토픽 2}**
   - {핵심 내용 요약}
   - 관련 참여자: {이름들}

### 핵심 메시지

- {작성자} ({시간}): {중요 메시지 요약}
- {작성자} ({시간}): {중요 메시지 요약}

### 질의응답

| 질문 | 답변 | 답변자 |
|------|------|--------|
| {질문 요약} | {답변 요약} | {답변자} |

### 액션 아이템

- [ ] {할 일} - {담당자}
- [ ] {할 일} - {담당자}

### 참여자 통계

| 참여자 | 메시지 수 |
|--------|:---------:|
| {이름} | {N}건 |
```

> 관련 없는 섹션은 생략 가능

## 4단계: 결과 출력 및 공유 선택

### 요약 결과 출력

분석 결과를 먼저 텍스트로 출력한다.

### 공유 여부 확인

AskUserQuestion으로 확인:
- "요약 결과를 어떻게 공유할까요?"
- 옵션:
  - "Slack 채널에 공유" (원본 채널에 요약 전송)
  - "다른 채널에 공유" (채널 선택)
  - "Confluence 저장" (Confluence 페이지로 생성)
  - "출력만" (추가 공유 없이 종료)

### Slack 공유 선택 시

도구: `mcp__claude_ai_Slack__slack_send_message`
- `channel_id`: 선택한 채널
- 메시지: 요약 내용

### Confluence 저장 선택 시

도구: `mcp__atlassian__create-page`
- `spaceId`: 1983741954 (개인 스페이스)
- `parentId`: 1983742135
- `title`: `[Digest] #{채널명} - {YYYY-MM-DD}`
- `bodyValue`: Confluence Storage Format (XML)

## 5단계: 결과 보고

```
## Slack Digest 완료

- **채널**: #{채널명}
- **기간**: {시작} ~ {끝}
- **메시지 수**: {N}건
- **주요 토픽**: {N}개
- **공유**: {공유 방식}
```
