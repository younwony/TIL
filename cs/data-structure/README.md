# Data Structure

자료구조 관련 학습 내용을 정리합니다.

## 목차

### 선형 자료구조
- [배열 (Array)](./array.md) - 연속 메모리 기반 자료구조, O(1) 접근
- [연결 리스트 (Linked List)](./linked-list.md) - 노드와 포인터 기반 자료구조, 동적 크기
- [스택과 큐 (Stack & Queue)](./stack-queue.md) - LIFO/FIFO 자료구조, 덱

### 비선형 자료구조
- [트리 (Tree)](./tree.md) - 계층 구조, 이진 트리, BST, AVL, Red-Black Tree
- [그래프 (Graph)](./graph.md) - 정점과 간선, 인접 행렬/리스트, DFS/BFS

### 해시 기반 자료구조
- [해시 테이블 (Hash Table)](./hash-table.md) - 해시 함수, 충돌 해결, O(1) 평균 시간

### 힙 기반 자료구조
- [힙 (Heap)](./heap.md) - 완전 이진 트리, 최대힙/최소힙, 우선순위 큐
