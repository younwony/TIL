# Productivity

일일 워크플로우 도구. 코드 외 작업(문서화, 상태 추적, 세션 관리, 회고)을 커버한다.

## Skills

- **[caveman](./caveman/SKILL.md)** — 응답에서 filler 제거. 토큰 75% 절약 모드.
- **[work-log](./work-log/SKILL.md)** — 브랜치 작업 내용을 Confluence에 자동 문서화.
- **[skill-rebuild](./skill-rebuild/SKILL.md)** — 기존 스킬 개선 시 skill-creator 전체 파이프라인 강제.
- **[sync-global](./sync-global/SKILL.md)** — 글로벌 설정과 프로젝트 .claude/ 동기화 (push/pull/status).
- **[setup-til-skills](./setup-til-skills/SKILL.md)** — 새 PC 클론 직후 hard dependency 점검 + 초기 세팅.
- **[track-status](./track-status/SKILL.md)** — Track 작업 추적 현황 + 6단계 워크플로우 가이드.
- **[track-status-summary](./track-status-summary/SKILL.md)** — Track 작업 요약 대시보드(MD + HTML) 생성.
- **[mermaid-diagram](./mermaid-diagram/SKILL.md)** — Mermaid CLI로 20가지 유형 다이어그램 생성.
- **[svg-diagram](./svg-diagram/SKILL.md)** — SVG 직접 생성 (정교한 픽셀 제어용).
- **[smart-session](./smart-session/SKILL.md)** — 세션 시작/실행/종료 라이프사이클 오케스트레이션 (WAT 프레임워크).
- **[weekly-retro](./weekly-retro/SKILL.md)** — 주간 git 활동 분석 → 학습/패턴/개선점 정리.
