# CS 문서 작성 규칙

CS 문서 작성 시 반드시 [CS-GUIDE.md](../../cs/CS-GUIDE.md)를 참조한다.

## 필수 요소

- **난이도 태그**: `[1] 정의` ~ `[4] 심화` 중 선택
- **선수 지식**: 링크로 연결 (없으면 "없음")
- **한 줄 정의**: 명확하고 간결한 정의
- **키워드**: 검색용 키워드 10~30개 (`#한글` `#English` 형식)
- **왜 알아야 하는가**: 실무/면접/기반 지식 관점
- **쉽게 이해하기**: 실생활 비유
- **면접 예상 질문**: "왜?"에 대한 답변 포함
- **연관 문서**: 선수/관련/후속 문서 테이블

## 다이어그램

- SVG 직접 생성 기본 (Mermaid는 빠른 프로토타이핑용)
- `cs/{category}/images/` 폴더에 저장
- 마크다운에서 `![설명](./images/{name}.svg)`로 참조
