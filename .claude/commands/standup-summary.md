# Standup Summary - 스탠드업 일일 요약

스탠드업 채널의 오늘 대화 내용을 분석하여 일일 요약을 생성해줘.

## 설정

| 항목 | 값 |
|------|------|
| **기본 채널** | `$ARGUMENTS`에서 지정 또는 사용자 선택 |

## 1단계: 채널 결정

### `$ARGUMENTS`에 채널 지정이 있는 경우

- `$ARGUMENTS`에서 채널명 또는 채널 ID를 추출
- 예: `/standup-summary #daily-standup` → `daily-standup` 채널 요약

### 채널 지정이 없는 경우

AskUserQuestion으로 채널을 요청:
- "스탠드업 채널을 입력해주세요. (채널명 또는 ID)"
- 옵션:
  - "채널 검색"
  - "채널 ID 직접 입력"

### 채널 검색 시

도구: `mcp__claude_ai_Slack__slack_search_channels`
- "standup" 또는 사용자 입력 검색어로 채널 검색
- AskUserQuestion으로 채널 선택

## 2단계: 채널 대화 읽기

도구: `mcp__claude_ai_Slack__slack_read_channel`

- `channel_id`: 1단계에서 결정한 채널 ID
- `response_format`: "detailed"

### 채널 읽기 실패 시

- "채널을 읽을 수 없습니다." 안내 후 중단

## 3단계: 스탠드업 내용 분석

읽어온 대화에서 오늘의 스탠드업 내용을 추출하고 분석한다.

### 스탠드업 분석 기준

각 참여자의 메시지에서 다음을 추출:

| 카테고리 | 키워드 |
|----------|--------|
| **어제 한 일** | 어제, 완료, 처리, 마무리, 진행했 |
| **오늘 할 일** | 오늘, 예정, 진행할, 시작할, 할 예정 |
| **블로커/이슈** | 블로커, 이슈, 문제, 지연, 도움, 확인 필요 |

### 요약 템플릿

```markdown
## 스탠드업 요약 - {YYYY-MM-DD}

**채널**: #{채널명}
**참여자**: {N}명 / 전체 {M}명

---

### 팀 현황 요약

| 구분 | 건수 |
|------|:----:|
| 어제 완료 작업 | {N}건 |
| 오늘 예정 작업 | {N}건 |
| 블로커/이슈 | {N}건 |

---

### 개인별 스탠드업

#### {참여자1}
- **어제**: {완료한 작업 요약}
- **오늘**: {예정 작업 요약}
- **블로커**: {있으면 표시, 없으면 "없음"}

#### {참여자2}
- **어제**: {완료한 작업 요약}
- **오늘**: {예정 작업 요약}
- **블로커**: {있으면 표시, 없으면 "없음"}

---

### 블로커 및 주의사항

| 참여자 | 블로커/이슈 | 긴급도 |
|--------|------------|:------:|
| {이름} | {블로커 내용} | {높음/보통} |

### 미참여자

{오늘 스탠드업에 참여하지 않은 멤버 목록 - 파악 가능한 경우}
```

> 블로커가 없으면 "블로커 및 주의사항" 섹션 생략

## 4단계: 결과 출력 및 공유

### 요약 출력

분석 결과를 텍스트로 출력한다.

### 공유 여부 확인

AskUserQuestion으로 확인:
- "스탠드업 요약을 어떻게 공유할까요?"
- 옵션:
  - "Slack 채널에 공유" (원본 채널에 요약 전송)
  - "다른 채널에 공유"
  - "출력만" (추가 공유 없이 종료)

### Slack 공유 선택 시

도구: `mcp__claude_ai_Slack__slack_send_message`
- `channel_id`: 선택한 채널
- 메시지 내용:

```
:clipboard: *스탠드업 요약 - {YYYY-MM-DD}*

*참여자*: {N}명

*완료 작업*: {N}건
*예정 작업*: {N}건
*블로커*: {N}건

{개인별 핵심 1줄 요약}
- {참여자1}: {오늘 핵심}
- {참여자2}: {오늘 핵심}

{블로커가 있는 경우}
:rotating_light: *블로커*
- {참여자}: {블로커 내용}
```

## 5단계: 결과 보고

```
## 스탠드업 요약 완료

- **채널**: #{채널명}
- **날짜**: {YYYY-MM-DD}
- **참여자**: {N}명
- **블로커**: {N}건
- **공유**: {공유 방식}
```
